# 商品管理システム

## 概要
このシステムでは店舗在庫の確認を権限をもつ人、持たない人にかかわらずWebシステム上で確認できるようにしたものです。
商品の登録・編集・削除は権限を持つものが行い、在庫の確認閲覧はユーザーであれば誰でも確認できるようにしました。

## 主な機能
- 管理者・一般ログイン・ログアウト機能
- 商品一覧画面
- 商品登録・編集・削除機能
- 商品検索機能

### 環境構築
PHP 8.1.10 
MySQL 5.6.50-log
Laravel Framework 8.83.3
